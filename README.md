# Reaction slider game description:
This a relatively new idea compared to games like tic tac toe or minesweeper, it is a slider game where a line moves from left to right on a red rectangle and a small green rectangle inside, the goal is to pause at the right time when the line hovers over the green rectangle, there is also the rare blue rectangle, wich is thinner and requires more presion but boasts more points and the legendary purple rectangle wich is even thinner and gives even more points, there are 4 difficulties each with customizable settings in the code eg. lives, line speed and progression. it does include level progression so that every 100 pts it gets harder. Finnaly there is an infinite mode to just train and grind, the score for the infinite mode does not affect any of your actual gameplay and you can go back to the menu with the "0" key.there is also a secret rainbow color for this aswell although due to a bug it looks ass because it flickers.
# How to install:
To use and import these into your calculator (Graph math+, but i belive also functions the same of the fx... version) first download the flies, then you plug the usb cable to the computer and select USB flash, then from the computer drag these files directly into the DISK E that will show. NOTE, do not put into MAINMEM or it will not work, then carefully eject or unplug from your computer FIRST before unpluging the calculator side and then open the python app on the calculator and press the TOOL button then OPEN and select the python file and run the code, may take 1-3 seconds to load.

# FOR SNAKE:
EXE-start

0-walls on or off

4- switch left in difficulty switcher

6- switch right in difficulty switcher

8- increase grid size

2- decrease grid size

OK- toggle snake colors (default green)

USE ARROWS TO CONTROL SNAKE

SECRET: SHIFT + OK - rainbow snake

# FOR FLAPPY BIRD:
4- switch left in difficulty switcher

6- switch right in difficulty switcher

EXE - play

7- toggle bird colors (default yellow)

9- toggle pipe colors (default green)

OK- jump, flaps the bird

# FOR REACTION GAME:
4- switch left in difficulty switcher

6- switch right in difficulty switcher

EXE- play

SHIFT + EXE- infinite mode

0- end infinite mode

5- toggle line color (default black)

OK- stop line

SECRET: SHIFT + 5 -rainbow line
